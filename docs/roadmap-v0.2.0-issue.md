# Roadmap: v0.2.0

QDB v0.2.0 focuses on reliability, observability, and operational maturity
while remaining a focused embedded queue library.

## Implemented after v0.1.0

- Configurable lease timeout API
- Queue statistics API

## Planned work

- Multi-process stress testing
- `qdb_compact()`

## Non-goals

- Networking
- Clustering
- Replication
- Language bindings
- Broker/server mode
