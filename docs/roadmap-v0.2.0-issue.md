# Roadmap: v0.2.0

QDB v0.2.0 will focus on reliability, observability, and operational maturity
while remaining a focused embedded queue library.

## Planned work

- Multi-process stress testing
- `qdb_compact()`
- Configurable lease timeout API
- Queue statistics API

## Non-goals

- Networking
- Clustering
- Replication
- Language bindings
- Broker/server mode
